## üåü Features

- **Fast & Lightweight** ‚Äì No JavaScript by default  
- **Component-Friendly** ‚Äì Works with React, Vue, Svelte, and more  
- **Server & Static Rendering** ‚Äì Choose what works for you  
- **Content-Driven** ‚Äì Markdown & MDX support built-in  
- **Extensible** ‚Äì Use integrations for Tailwind, Sitemap, and more  

---

## üöÄ Installation

The **recommended** way to install Astro:

```bash
npm create astro@latest
```

For a manual installation:

```bash
npm install --save-dev astro
```

- **[Getting Started Guide](https://docs.astro.build/en/getting-started/)**
- **[Try a Starter Project](https://astro.new/)**
